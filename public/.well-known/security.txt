# Security Disclosure Policy
# If you discover a security vulnerability, please report it to security@riqle.com
# We take security seriously and will respond to valid reports within 48 hours.

# Please include:
# - Description of the vulnerability
# - Steps to reproduce
# - Potential impact
# - Any proof-of-concept code (if applicable)

# Out of Scope:
# - Social engineering attacks
# - Physical attacks
# - Denial of Service (DoS/DDoS)
# - Spam or brute-force attacks
# - Issues in third-party services (e.g., Stripe, email providers)

# Acknowledgments
# We appreciate responsible disclosure and will credit researchers who report valid vulnerabilities.
